# bpool
A Go (Golang) lib to manage buffers with a constant size with buffer reuse and garbage collector.


## Get bpool 

```bash
go get github.com/90TechSAS/go-buffer-pool-recycler
```

## Update bpool 

```bash
go get -u github.com/90TechSAS/go-buffer-pool-recycler
```
